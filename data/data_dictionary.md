# Data Dictionary: E-Commerce Star Schema [goldie]

This document serves as the central data governance reference manual for the production business intelligence pipeline. It outlines definitions, primary keys, foreign keys, and internal operational data constraints across the relational schema hubs.

---

## 1. Table: `fact_sales`

| Column Name | Data Type | Key Type | Description / Business Rule |
| :--- | :--- | :--- | :--- |
| `order_number` | `VARCHAR(20)` | PK | Unique identifier for each distinct customer transaction invoice. |
| `order_date` | `DATE` | FK | Date the transaction was finalized. Anchor for macro time trend calculations. |
| `shipping_date` | `DATE` | - | Date the inventory left the fulfillment center. Must be $\ge$ `order_date`. |
| `customer_key` | `INT` | FK | Relates to `dim_customers`. Used to calculate discrete customer RFM scores. |
| `product_key` | `INT` | FK | Relates to `dim_products`. Grain base used to compute the 80/20 Pareto distribution logic. |
| `quantity` | `INT` | - | Total unit count of items purchased per line item transaction. Value must be > 0. |
| `sales_amount` | `DECIMAL(18,2)`| - | Gross transactional revenue generated by the line item sale, exclusive of currency conversions. |

## Key Pipeline Transformation Metrics

* **Fulfillment SLA Calculation:** Computed as `DATEDIFF(DAY, order_date, shipping_date)`. If the resulting latency delta exceeds 7 days, the system automatically classifies the log record as a supply chain bottleneck.
* **Automated Customer RFM Personas:** Engineered by processing recency, frequency, and monetary fields through independent `NTILE(5)` window arrays. The resulting codes are mapped into strategic operational classes via standardized relational lookup scripts (e.g., *Champions*, *At Risk*).
* **Pareto Tier Classifications:** Derived via rolling cumulative profit metrics using a deterministic window ordering function: `SUM(Profit) OVER (ORDER BY Profit DESC, product_key)`. Items maintaining the running balance under the strict 80% boundary line are flagged as *Top Tier Assets*.
